API列表:

注册
HTTP Method: POST
地址:/register
参数:username(string) string(string)
成功的返回:{ "token" : "8bc5e768ca33418281900662f1f788d3" }
 
登录
HTTP Method: POST
地址:/login
参数:username(string) string(string)
成功的返回:{ "token" : "8bc5e768ca33418281900662f1f788d3" }

修改密码
HTTP Method: PUT
地址:/password/change
参数:oPassword(string) nPassword(string)
成功的返回:空

地址: /profile
